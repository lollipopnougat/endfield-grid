# 支持新设备需求

## 仓库取货口

3x1尺寸 颜色为浅灰色

其输出口位于中间格子上，如下所示
```
-
>
-
```
“-”表示非输入输出口

要求与现在设备展示形式类似，渲染指示输出箭头位于其输出口上，此设备不需要供电，无需检测供电条件展示图标

## 仓库存货口

```
-
<
-
```

要求与现在设备展示形式类似，渲染指示输入箭头位于其输入口上，此设备不需要供电，无需检测供电条件展示图标

## 协议核心

9x9尺寸 白色

这是一个特殊的设备，有14个输入口和6个输出口，输入口对称分布，输出口也对称分布，其结构如下所示
```
-↑--↑--↑-
>-------<
>-------<
>-------<
>-------<
>-------<
>-------<
>-------<
-↓--↓--↓-
```
“-”表示非输入输出口，“>”和“<”代表了输入口，并指向了其输入方向，“↑”和“↓”代表了输出口，并指向了其方向

要求与现在设备展示形式类似，渲染指示输入箭头位于其上，此设备不需要供电，无需检测供电条件展示图标
